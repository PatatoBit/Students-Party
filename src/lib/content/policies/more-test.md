# จัดการทดสอบความรู้และกิจกรรมทางด้านวิชาการในแต่ละหมวดวิชา
นักเรียนแต่ละคนมีความถนัดในแต่ละหมวดวิชาไม่เหมือนกันเราจึงจัดการทดสอบความรู้ให้นักเรียนทดลองสอบ เพื่อวัดประเมินตัวเอง รวมถึงมีการแข่งเขียนเรียงความและEssay(ตามวันสําคัญ, การประกวดแต่งกลอน(ตามวันสําคัญ) และมีการแข่งตอบคําถามอีกด้วย โดยที่จะมีใบประกาศและของรางวัลให้ด้วย
