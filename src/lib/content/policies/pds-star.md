# ยกเลิกเพลงเด็กเอ๋ยเด็กดี เปลี่ยนเป็นดาวเด่นปทุมวัน
PDS Wave จะกล่าวแสดงความชื่นชมกับนักเรียนที่สร้างผลงานให้แก่โรงเรียนในช่วงเช้า โดยนักเรียนสามารถมาเสนอตัวเองพร้อมแนบหลักฐาน เพื่อที่จะให้ PDS Wave มาประกาศในวันข้างหน้าได้ โดยจะเอาเวลานี้มาจากการยกเลิกการเปิดเพลงเด็กเอ๋ยเด็กดี
